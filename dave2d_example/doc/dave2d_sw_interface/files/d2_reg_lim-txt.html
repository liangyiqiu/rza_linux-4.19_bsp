Limiter registers

All limiter registers are write only, reading will return undefined results.  Limiters must be enabled (see D2_CONTROL) in order to produce an output value.

In addition to the 6 geometry limiters there are two special UV-Limiters (see Texture registers).

Banding

Postfilter

First two limiter outputs can be routed through an additional unit before clamping in order to mirror their gradient and form a ‘band’ instead of the usual ‘halfplane’.

D2_L1BAND

Limiter1 band width parameter

Offset: 22

When band ouput filtering is enabled for limiter1 (D2C_BAND1ENABLE see D2_CONTROL) this register stores the 16:16 fixedpoint inner width of band region.
